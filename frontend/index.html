AX - STUDIO HELP

AX - Studio is a dashboard that interacts with our hybrid product attribute extraction server.

1. Select one of the prefilled pill buttons to load an evaluation seed sentence from the assignment split, or type custom description text.

2. Choose the extraction strategy. **ML** uses the hybrid classification pipeline with automatic low-support fallbacks. **Rule-Based** executes literal vocabulary matching.

3. Click **Extract Attributes** to run predictions. Output cards highlight parsed values and confidence percentages.
